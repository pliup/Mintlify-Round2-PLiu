The Deepgram API is a RESTful API that allows you to interact with Deepgram programmatically. You can use our API to:
* Transcribe pre-recorded and streaming audio.
* Transform text to speech.
* Analyze text.
* Admini9ster your Deepgram account:
  * Manage projects and project members
  * Manage project invitations
  * Manage user scopes
  * Obtain billing balances
  * Obtain usage summaries
  * Manage API keys
  * Manage On-Prem credentials
